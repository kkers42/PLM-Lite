# PLMLITE

PLMLITE is an open-source Product Lifecycle Management (PLM) application for managing Siemens NX12 CAD datasets. It provides tools for tracking files, metadata, revisions, and lifecycle states within Python environments. 

## Features

- Parsing and ingesting NX12 CAD files
- Metadata extraction and indexing
- Lifecycle state management (e.g., design, review, release)
- Command-line interface for simple operations

## Getting Started

Install with pip:

```bash
pip install plmlite
```

Or from source:

```bash
git clone https://github.com/yourusername/PLMLITE.git
cd PLMLITE
pip install -e .
```
